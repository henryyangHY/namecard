# Henry Yang — Virtual Name Card

A minimal, single-page personal landing page.  
Live: [henryyanghy.github.io/namecard](https://henryyanghy.github.io/namecard/)

## Stack

- Pure HTML + CSS + vanilla JS
- Hosted on GitHub Pages (no build step)
- Fonts: [Outfit](https://fonts.google.com/specimen/Outfit) + [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono)

## Features

- Click avatar → download vCard (`.vcf`) for quick contact saving
- Responsive layout (mobile-first)
- SEO-optimized with Open Graph + Twitter Card + JSON-LD
- Lightweight (~50KB total excluding photo)

## How to Update

1. Edit `index.html` for content changes
2. Edit `assets/css/v2.css` for styling
3. Replace `images/avatar.jpg` for photo
4. Edit `images/henry-yang.vcf` for contact info
5. Push to `main` — GitHub Pages deploys automatically
